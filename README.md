# siftday

Music platform using React, TS, SCSS, Tailwind, Chakra UI, Redux Toolkit, RTQ Query, React Hook Form

## Run Locally

Clone the project

```bash
  git clone https://github.com/sifmeop/siftdos.git
```

Go to the project directory

```bash
  cd siftdos
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
